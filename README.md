# 🖥️ SolucionesDigitales

> Catálogos web profesionales y soluciones IT para emprendimientos en Florida, Uruguay.

🌐 **Sitio en vivo:** [solucionesdigitales-uy.vercel.app](https://solucionesdigitales-uy.vercel.app)

---

## 📋 Descripción

**SolucionesDigitales** es un sitio web orientado a emprendedores y pequeñas empresas de Florida, Uruguay, que buscan tener presencia digital profesional. Ofrecemos catálogos web modernos, mantenimiento de sistemas Windows e instalación de software y aplicaciones.

---

## 🚀 Servicios

- **Catálogos Web Modernos** — Interfaces rápidas, diseño responsive y optimización SEO.
- **Mantenimiento Pro** — Optimización avanzada de sistemas Windows para máximo rendimiento.
- **Software & Apps** — Instalación y configuración de herramientas esenciales para tu flujo de trabajo.
